# 🔐 Password Strength Checker

## SkillCraft Technology Cyber Security Internship - Task 3

## Project Overview

This project focuses on developing a **Password Strength Checker using Python** as part of the **SkillCraft Technology Cyber Security Internship - Task 3**.

The tool evaluates the strength of a password based on important security parameters such as password length, uppercase letters, lowercase letters, numbers, and special characters.

The program analyzes the password, calculates a strength score, classifies the password as Very Weak, Weak, Moderate, Strong, or Very Strong, and provides suggestions to improve password security.

## Working Principle

The Password Strength Checker evaluates passwords using multiple security criteria.

### Password Validation Criteria:

- **Length Check**
  - Ensures the password contains at least 8 characters.

- **Uppercase Letter Check**
  - Verifies the presence of capital letters (A-Z).

- **Lowercase Letter Check**
  - Verifies the presence of small letters (a-z).

- **Number Check**
  - Checks whether the password contains numerical digits (0-9).

- **Special Character Check**
  - Detects symbols such as @, #, $, %, and other special characters.

Each satisfied condition increases the password score. Based on the final score, the password strength is displayed.
